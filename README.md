# 🚀 Personal Portfolio Website

A modern, responsive portfolio website built with vanilla HTML, CSS, and JavaScript. This portfolio showcases my skills, projects, and professional journey in web development.

## ✨ Features

### 🎯 Clean and Modern Design
- Minimalist and professional layout
- Smooth animations and transitions
- Fully responsive design for all devices
- Dark theme with elegant color scheme

### 🛠️ Skills Section
- Interactive skill cards with progress bars
- Categorized skills (Frontend, Backend, Tools)
- Visual representation of proficiency levels
- Animated on scroll effects

### 💼 Projects Showcase
- Project cards with descriptions
- Technology tags for each project
- Links to live demos and source code
- Image previews for projects

### 🎨 Design Elements
- Custom color palette
- Font Awesome icons
- AOS (Animate On Scroll) animations
- Hover effects and transitions

## 🎨 Color Palette
- Primary: `#0A192F` (Dark Blue)
- Secondary: `#64FFDA` (Teal)
- Text Primary: `#CCD6F6` (Light Blue)
- Text Secondary: `#8892B0` (Muted Blue)

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome (icons)
- AOS Library (animations)

## 📱 Responsive Design
- Mobile-first approach
- Breakpoints at 768px and 480px
- Flexible grid layouts
- Optimized for all screen sizes

## ⚡ Performance
- Minimal JavaScript usage
- Optimized images
- Efficient CSS animations
- Fast loading times

## 🚀 Quick Start
1. Clone the repository
   ```bash
   git clone https://github.com/adeshsk18/portfolio.git
   ```
2. Open `index.html` in your browser
3. Customize the content in HTML files
4. Modify styles in `css/style.css`
